# SE-TheLostItemFinder 물건을 찾아줘

## 프로젝트 소개

학교, 회사등에서 분실물이 발생했을 경우 분실한 물건을 찾기위해 돌아다녀야하는 불편함을 줄이고
각 시설담당자가 보다 편리하게 업무처리를 할 수 있도록 돕는 웹사이트입니다.
